# Mesxent Global Stratum Pool Engine
**Organization ID:** 11270629836102

## Overview
Mesxent Global Stratum is a high-performance, policy-driven mining pool engine designed for large-scale resource allocation. It integrates real-time financial data validation and automated revenue distribution.

## Key Features
* **Automated Payout Policy:** Hardcoded 70/20/10 revenue split.
* **Profit-Switching Intelligence:** Powered by real-time market API integration.
* **Agro-Node Connectivity:** Direct link to Mesxent physical asset nodes.

## Quick Start
1. Clone the repository: `git clone https://github.com/jamosj/mesxent-stratum-pool.git`
2. Install dependencies: `npm install`
3. Configure your keys in `.env` (Use `config_example.json` as a template).
4. Launch the engine: `node init.js`

## Security & Policies
This repository follows strict "Proof of Work" validation. All transactions are verified via on-chain node protocols to ensure subscriber transparency.
